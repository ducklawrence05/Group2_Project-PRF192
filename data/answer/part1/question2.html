Vẽ hình chữ nhật
Source
#include <stdio.h>
int main() {
  int length, width;
  printf("Nhập chiều dài và chiều rộng của hình chữ nhật: ");
  scanf("%d", &length);
  scanf("%d", &width);
  for (int i = 0; i < length; i++) {
      for (int j = 0; j < width; j++) {
          if (i == 0 || i == length - 1 || j == 0 || j == width - 1) {
              printf("* ");
          } else {
              printf("  ");
          }
      }
      printf("\n");
  }
  return 0;
}

Giải thích

  • Dòng đầu tiên và dòng cuối cùng: In toàn bộ dấu *.
  • Các dòng ở giữa: Chỉ in * ở cột đầu tiên và cột cuối cùng, khoảng trống ở giữa.

Và khi nhập input là 5 3 chúng ta sẽ có output như bên dưới:

Output
* * * * *
*       *
* * * * *
