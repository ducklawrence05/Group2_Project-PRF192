Kiểm tra số nguyên tố

Số nguyên tố là gì?

Số nguyên tố là số tự nhiên lớn hơn 1 chỉ có 2 ước số là 1 và chính nó. 

Source
#include <stdio.h>
#include <math.h>
       
int main() {
  int n, isPrime = 1;

  printf("Nhập một số nguyên: ");
  scanf("%d", &n);

  if (n < 2 || (n % 2 == 0 && n > 2)) {
      isPrime = 0; // Số nhỏ hơn 2 hoặc số chẵn lớn hơn 2 không phải số nguyên tố
  } else {
      for (int i = 3; i <= sqrt(n); i += 2) { // Chỉ kiểm tra số lẻ
          if (n % i == 0) {
              isPrime = 0;
              break;
          }
      }
  }

  if (isPrime) {
      printf("%d là số nguyên tố.\n", n);
  } else {
      printf("%d không phải là số nguyên tố.\n", n);
  }

  return 0;
}
Input
Nhập một số nguyên: 5
Output
5 là số nguyên tố.

Về số 2

  • Số 2 là số chẵn duy nhất là số nguyên tố, khi 2 qua được điều kiện if và tới else, nó sẽ không thể vào vòng lặp do điều kiện bắt đầu của vòng lặp là i = 3 và kết thúc là i <= sqrt(n)
  • Do đó mà isPrime vẫn giữ giá trị 1.
