Tìm tích gồm 2 thừa số của 1 số sao cho độ chênh lệch của 2 thừa số là nhỏ nhất

Ý tưởng

  • Độ chênh lệch giữa hai thừa số nhỏ nhất khi hai thừa số gần nhau nhất.
  • Điều kiện để hai số là thừa số của một số n là n % i == 0 (tức là i chia hết n, và thừa số còn lại là n / i).
  • Để đảm bảo độ chênh lệch nhỏ nhất, duyệt từ sqrt(n) ngược về 1, vì các thừa số gần nhau nhất sẽ nằm gần căn bậc hai của n. 
Source
#include <stdio.h>
#include <math.h>
         
int main() {
  int number;
  printf("Nhập số nguyên cần xử lý: ");
  scanf("%d", &number);

  int numberAbs = abs(number); // Lấy giá trị tuyệt đối để tránh input là số âm
  int factor1 = 1, factor2 = numberAbs; // Khởi tạo thừa số
  int minDifference = numberAbs - 1; // Khởi tạo độ chênh lệch lớn nhất
  
  if(number != 0){
    for(int i = sqrt(numberAbs); i >= 1; i--){
      if(numberAbs % i == 0){ // Nếu i là thừa số
        int j = numberAbs / i;
        if(abs(i - j) < minDifference){ // Nếu chênh lệch nhỏ hơn
          factor1 = i;
          factor2 = j;
          minDifference = abs(i - j);
        }
      }
    }
  
    // Nếu số ban đầu là âm, gán dấu âm cho một thừa số
    if(number < 0){
      factor1 *= (-1);
    }
  }else{
      // Nếu số là 0, cặp thừa số là 0 và 0
      factor1 = 0;
      // factor2 đã gán bằng số ban đầu nên không cần gán 0 cho factor2 nữa
  }

  printf("Hai thừa số cần tìm là %d và %d", factor1, factor2);

  return 0;
}
Input
Nhập số nguyên cần xử lý: -28
Output
Hai thừa số cần tìm là: -4 và 7
